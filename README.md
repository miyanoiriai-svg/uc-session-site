# UC Session

UCセッションのブランドサイト（https://uc-session-since2017.netlify.app）のソースコードです。

## 現在の構成（ステップ1: Git移行のみ）

- `index.html` … サイト本体（静的HTML、ビルド不要）
- `robots.txt` … クローラー向け設定
- `sitemap.xml` … サイトマップ
- `og-image.jpg` … SNSシェア用画像
- `netlify.toml` … Netlifyのデプロイ設定（ビルドコマンドなし、公開ディレクトリはルート）

このリポジトリをNetlifyの既存サイトに連携するだけで、
現在ドラッグ&ドロップでデプロイしている内容と完全に同じものが公開されます。
デザイン・機能の変更はありません。
